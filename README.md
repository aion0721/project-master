# 案件管理Webアプリ MVP

React + TypeScript + Vite で作成した、案件管理向けのフロントエンドMVPです。  
バックエンドは持たず、モックデータだけで案件一覧、案件詳細、複数案件横断ビューを確認できます。

## 実装済み機能

- 案件一覧画面
  - 案件名、現在フェーズ、PM、状態、開始日、終了予定日を一覧表示
  - 状態を色付きバッジで表示
  - 詳細画面への遷移
- 案件詳細画面
  - 案件情報、PM、期間、OS担当を表示
  - 週単位のフェーズ進捗タイムライン
  - フェーズ別担当者テーブル
  - 上司・部下関係を含むプロジェクト体制ツリー
- 複数案件横断ビュー
  - 案件ごとの週別フェーズを横断表示
  - フェーズ重複週を確認しやすいUI

## 技術構成

- React 19
- TypeScript
- Vite
- react-router-dom
- CSS Modules
- Context + モックデータによる軽量なフロント管理

## 起動方法

```bash
yarn
yarn dev
```

ブラウザで `http://localhost:5173` を開いてください。

## ビルド

```bash
yarn build
```

